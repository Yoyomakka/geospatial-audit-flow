# geo-audit-flow Documentation

`geo-audit-flow` is a small, config-driven geospatial workflow tool for data QA, cleaning, lightweight spatial analysis, map previews, and reports.

The project philosophy is intentionally practical:

- Make common geospatial QA repeatable.
- Keep workflows readable in YAML.
- Prefer small, transparent outputs over hidden state.
- Run locally on a normal laptop.
- Use synthetic examples that are safe to share.

It is not a web app, database server, cloud pipeline, or massive-data processing framework. The first version focuses on dependable vector data workflows for students, analysts, and portfolio projects.
